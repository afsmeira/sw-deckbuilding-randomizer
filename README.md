# SW Deckbuilding Game Randomizer

A faction and galaxy deck randomizer for [Star Wars: The Deckbuilding Game](https://www.fantasyflightgames.com/en/products/star-wars-the-deckbuilding-game/) by Fantasy Flight Games.

**[Try it live](https://afsmeira.github.io/sw-deckbuilding-randomizer/)**

## Features

- Randomly assigns factions to each player
- Randomly selects a galaxy deck
- Supports multiple products and game modes

### Supported products

| Product | Factions | Galaxy deck |
|---|---|---|
| Star Wars: The Deckbuilding Game (Base Game) | Galactic Empire, Rebel Alliance | Base Game galaxy deck |
| Star Wars: The Deckbuilding Game — Clone Wars Edition | Galactic Republic, Separatist Confederacy | Clone Wars galaxy deck |
| Mandalorian Faction Pack *(expansion)* | Mandalorian | Mandalorian 3-player galaxy deck |

### Supported game modes

- **1v1** — two players, each assigned a faction and a shared galaxy deck
- **2v2** — four players across two regions, each region with its own galaxy deck
- **3-Player Free-for-All** — three players (requires the Mandalorian Faction Pack)

## Running locally

No build step required. Clone the repository and open index.html in a browser:

```bash
git clone https://github.com/afsmeira/sw-deckbuilding-randomizer.git
cd sw-deckbuilding-randomizer
open index.html
```
